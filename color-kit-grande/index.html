The Color Kit Grande is an engaging and educational project suitable for students of all ages. It provides an opportunity to learn about electronics, programming and to enhance soldering skills. The kit includes all the components required to assemble a working weather station:
  • ESP32 ePulse Feather Development Board
  • 3,5‚Ä≥ Color Display with Capacitive Touch Screen
  • PCB to Connect Display and Microcontroller
  • Grove connector for various Extension Boards
  • On/Off Switch
  • Sticker Pads to glue the Display
  • Female and male header pins

The Color Kit Grande is perfect for absolute beginners, hobbyists, and STEM students!

To assemble the kit, you'll need a soldering iron, solder, a USB-C cable, and a computer to flash the firmware onto the ESP32. The supported operating systems include Windows, MacOS (Intel and Apple Silicon), and Linux. The kit includes all the necessary components to assemble the working hardware.

The kit includes a pre-configured weather station sample project, making it accessible even to beginners with no prior programming experience. You can build upon this foundation by modifying the provided code or creating your own projects, making it an excellent tool for both learning and advancing programming skills.
