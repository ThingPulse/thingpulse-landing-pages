FAQ

Have questions? We’ve got answers.

What is the Color Kit Grande?

The Color Kit Grande is an engaging and educational project suitable for students of all ages. It provides an opportunity to learn about electronics, programming and to enhance soldering skills. The kit includes all the components required to assemble a working weather station:

  • ESP32 ePulse Feather Development Board
  • 3,5″ Color Display with Capacitive Touch Screen
  • PCB to Connect Display and Microcontroller
  • Grove connector for various Extension Boards
  • On/Off Switch
  • Sticker Pads to glue the Display
  • Female and male header pins
Is The Color Kit Grande Suitable For Me?

The Color Kit Grande is ideal for absolute beginners, electronics hobbyists, and STEM students alike. It’s designed to be accessible and engaging, whether you’re just starting out or looking to expand your skills in electronics and programming.

Which tools/devices are required to assemble the kit?

To assemble the kit, you will need:

  • A soldering iron and solder
  • A USB-C cable
  • A computer to flash the firmware onto the ESP32 (compatible with Windows, macOS [Intel and Apple Silicon], and Linux)
The kit includes all the necessary components for building the working hardware.

Which programming skills are required?

The kit includes a pre-configured weather station project, so even if you're new to programming, you'll have a guided introduction. As you grow more confident, you can modify the code or create your own projects, making it perfect for both beginners and those looking to expand their programming skills.
